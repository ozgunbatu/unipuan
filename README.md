# UniPuan - Üniversite Sınav Puan Hesaplama

Türkiye'deki üniversite sınavları için kapsamlı puan hesaplama platformu.

## Özellikler
- YKS/TYT Puan Hesaplama
- KPSS Puan Hesaplama (P3, P4, P10)
- YDS/e-YDS Puan Hesaplama
- ALES Puan Hesaplama
- 2025 Üniversite Taban Puanları

## Kurulum
\`\`\`bash
npm install
npm run dev
\`\`\`

## Deploy
\`\`\`bash
npm run build
vercel deploy
\`\`\`
